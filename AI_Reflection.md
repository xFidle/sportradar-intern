#### Artifical Inteligence usage 
For backend development, I primarly used external chatbots such as ChatGPT and
Gemini to bounce off some ideas and refine architectual concepts. Code was
written manually. On the frontend side, I used [Opencode](https://opencode.ai/)
with GPT-5.3 Codex to scaffold the application and handle the inital setup.
Although much of the frontend code was generated, I still needed to make
numerous adjustments in the JavaScript files to achieve the desired behavior.
The CSS, however, was is entierly AI-generated - I didn't write a single line
by hand.
